Detailed Description

template<typename T, typename allocator = std::allocator<T>>
class Sequence::util::vectorizer< T, allocator >

Facilitates safe inheritance from std::vector<T, allocator> Privately inherits from std::vector and makes the public interface of that class accessible via using declarations.

Classes may safely inherit publicly from this class with no fear of memory leaks, etc.

Note
The interface is the C++11 interface to std::vector
