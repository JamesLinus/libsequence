size_t Sequence::bamaux::size
mutable

The size, in bytes, of value. If value_type is a string type, the length of the string stored in value will be size + 1 due to \0 padding at the end.

Note
size will be set to 0 If the programmer requests the an aux data field from and alignment, and that data field does not exist.
